## プロジェクト概要
制作時期 : 2023年 

**video-chat**は、**node.js**を用いて開発したビデオチャットWebアプリです。

**一対一**のリアルタイム通信を**P2P**で行い、**チャット**と**ビデオ通話**ができます。

## 使用方法
1. ルームタブで、任意のルーム名と名前を入力し、ルームに参加ボタンを押します。
2. 同じルーム名で参加したユーザーとマッチングします。
3. チャットやビデオ通話を楽しみます。

&nbsp;
